# atcoder-testcase-downloader

AtCoder用のChrome拡張機能です。

AtCoderの提出ページ内に公開済みテストケースのコピー機能を追加します。  
Dropboxからダウンロードする手間を省くことができます。  
ブラウザ上でリンクを辿ってダウンロードするのと同じことをしています。
